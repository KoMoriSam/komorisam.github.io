## 6. 代码块（Code Blocks）

### 行内代码

### 多行代码块


```javascript

```

## 12. HTML 支持（HTML Support）

可以直接在 Markdown 中使用 HTML，例如：

```html
<div style="color: red;">这是一段红色文字。</div>
```
